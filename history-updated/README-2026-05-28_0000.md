# Update 2026-05-28 00:00

## Scope
- Built a small-scale inventory MVP for Android-first use.

## What is included
- Inventory list with add, edit, and delete actions.
- Item fields: name, category, quantity, unit cost, optional expiry date.
- Summary card for total quantity and stock value.

## Next upgrade ideas
- Persist inventory locally (SQLite).
- Low-stock alerts and expiry warnings.
